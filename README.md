# learning-app

# how to run 

```
14x version npm
curl -sL https://deb.nodesource.com/setup_14.x | sudo -E bash -
sudo apt install nodejs
node -v 
sudo npm install --global yarn
yarn install 
```
# how to contribute

```
1. cd server/
2. check code: 
    yarn lint:fix
3. format code: (prettier)
    yarn format
4. Run dev:
    yarn dev
5. Run production:
   yarn start
```
